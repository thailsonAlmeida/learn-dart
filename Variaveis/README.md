# Variaveis

**Como é sintaxe da declaração da variavel?**
``` [modificador] tipo nome = <valor>  ```

**Quais são os modificadores das variveis?**
- late
- const
- final

**O que é o modificador late?**
- A variavel será inicializada posteriormente.

**O que é o modificador const**
- A variavel é definida em tempo de compilação

**O que é o modificador final**
- A variavel é deinifida apenas uma vez. É imutavel

**O que são variaveis do tipo String?**
- São objetos.
- Tem propriedades.
- Tem atibutos.

**Quais os principais atributos do tipo String?**
- hashCode
- isEmpty
- isNotEmpty
- length

**Quais os principais metodos do tipo String?**
- compareTo
- contains
- endsWith
- indexOf
- replaceAll
- replaceRange
- split
- startsWith
- substring
- toLowerCase
- toUpperCase
- trim
- trimLeft
- trimRight
